# val and var

```scala    
val immutable: Int = 42
var mutable: String = "I can change"
```

## Types

- Any
    - AnyVal
        - Float, Double
        - Byte, Short, Int, Long
        - Boolean
        - Char
        - Unit
    - AnyRef (`java.lang.Object`)
        - List
        - Option
        - YourClass
